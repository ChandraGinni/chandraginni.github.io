Android application

Its an application to drop a marker on google maps using longitude and latitude as parmameters.
When the application starts the first screen is a login activity.
If the user is not registerd, the user is redirected to register page activity when clicked on register.
After the user is successfully registered, user is redirected to the login.
When login is successful, the application will show a list of city names which are added before hand and displayed using list view.
In the menu section the user has an option to to add a city to the list.
To add a city the user must enter a city name, latitude and longitude values. once you click on add the city is added to the list.
